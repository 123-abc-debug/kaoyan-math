## 多元函数的基本概念

# **为什么？——连续的定义三条件**

一个函数 ( f(x,y) ) 在点 ( (x_0,y_0) ) 连续，当且仅当同时满足：

1. **函数在该点有定义**
   [
   f(x_0,y_0)\ \text{必须存在}
   ]

2. **极限存在**
   [
   \lim_{(x,y)\to(x_0,y_0)} f(x,y)\ \text{存在}
   ]

3. **极限等于函数值**
   [
   \lim_{(x,y)\to(x_0,y_0)} f(x,y)=f(x_0,y_0)
   ]

连续 = 三个条件 **缺一不可**。

---

# **你的问题属于：函数值不存在 → 直接“卡死”第一条条件 → 一定不连续**

即使（假设）你证明了
[
\lim_{(x,y)\to(0,0)} f(x,y)=0
]
但因为 **原点没有定义函数值 (f(0,0))**，所以连续性第一条不满足，仍然不连续。

就像一元函数中：
[
f(x)=\frac{\sin x}{x},\ x\ne 0
]
虽然极限存在（=1），但因为 (f(0)) 未定义，所以在 (x=0) **不连续**。

---

# **再进一步**

如果你愿意，你可以把
[
f(0,0)=0
]
**人为补上**，那么这个函数在原点是否连续才取决于极限是否为 0。

但在你给的函数中：

[
\lim_{(x,y)\to(0,0)} \frac{xy}{\sin(x^2+y)}
]
极限实际上 **不存在**（上条回答已证明），所以就算你补上 0，也不会连续。

# 全微分
## 1.全微分的定义
几何直观理解 某个点的局部是否光滑，能用一个平面去逼近吗


# 1️⃣ 问题出在哪里？

你看到的这个式子看起来很抽象：

[
f(x_0+h, y_0+k) = f(x_0, y_0) + A h + B k + o(\sqrt{h^2+k^2})
]

* (A) 和 (B) 是常数（实际上就是偏导）
* (o(\sqrt{h^2+k^2})) 是误差项，看起来“神秘”

很多人看了就蒙：为什么这样定义？为什么要用这个小 (o)？

---

# 2️⃣ 几何直观理解

假设你有一个函数 (f(x,y)) 的图像：

* 在点 ((x_0, y_0)) 画出来
* 我们想问：这个点**局部是否光滑**？是否能用一个平面去“逼近”？

➡️ “逼近平面”的思想就是：

> 曲面在点 ((x_0, y_0)) 附近，可以用一个平面来近似，这个平面就是切平面

切平面方程：

[
z = f(x_0,y_0) + f_x(x_0,y_0) (x-x_0) + f_y(x_0,y_0)(y-y_0)
]

* (f_x, f_y) 是偏导
* 这就是式子里的 (A h + B k)，其中 (h = x-x_0, k = y-y_0)

---

# 3️⃣ o((\sqrt{h^2+k^2})) 是什么意思？

* 它表示**比距离更小的高阶误差**
* (\sqrt{h^2+k^2}) 是点 ((x_0+h, y_0+k)) 到 ((x_0, y_0)) 的距离
* (o(\sqrt{h^2+k^2})) 意味着：

[
\lim_{(h,k)\to (0,0)} \frac{o(\sqrt{h^2+k^2})}{\sqrt{h^2+k^2}} = 0
]

📌 几何直观：

* 这个误差项比距离小 → 随着你靠近 ((x_0,y_0))，曲面离切平面的高度差 → 远小于你走的水平距离
* 换句话说：切平面逼近得越来越精确，曲面在局部“看起来就是平的”

## 3.全微分的形式不变性

## 4.可微判定方法
如何判定可微

# 3️⃣ 判定可微的方法

对于函数 (f(x,y))：

1. **直接用定义法（严格判定）**

   * 计算
     [
     \lim_{(h,k)\to(0,0)} \frac{f(x_0+h, y_0+k)-f(x_0,y_0)-f_x h - f_y k}{\sqrt{h^2+k^2}}
     ]
   * 如果极限存在且为 0 → 可微
   * 极限 = 0 的意思：高阶误差比距离小，切平面逼近

2. **偏导连续法（常用简便法）**

   * 如果偏导在点附近连续 → 可微 ✅

3. **方向导数法（慎用）**

   * 方向导数存在 **不保证可微**
   * 必须检查是否可以拼成切平面

---

# 4️⃣ 一句话总结

> **多元函数可微 = 局部能被一个平面精确逼近，高阶误差比距离小**
>
> * 局部光滑 → 切平面存在 → 函数连续
> * 判定可微 → 看定义或偏导连续性

我们用一个具体的 (f(u,v)) 来演示 **多元复合函数偏导 + 链式法则** 的计算过程，让你从符号理解过渡到具体计算。

---

# 1️⃣ 例子设定

假设：

[
z = f(u,v) = u^2 + \sin(v)
]

复合关系：

[
u = xy, \quad v = \frac{y}{x}
]

我们要算：

[
\frac{\partial z}{\partial x}, \quad \frac{\partial z}{\partial y}
]

---

# 2️⃣ 先求外函数的偏导数

[
f_u = \frac{\partial f}{\partial u} = 2u, \quad f_v = \frac{\partial f}{\partial v} = \cos(v)
]

注意：这里 (f_u, f_v) 还要代入 (u = xy, v = y/x) 后才能得到最终值。

---

# 3️⃣ 求内函数对 (x,y) 的偏导

[
u = xy \implies u_x = y, \quad u_y = x
]

[
v = \frac{y}{x} \implies v_x = -\frac{y}{x^2}, \quad v_y = \frac{1}{x}
]

---

# 4️⃣ 链式法则计算偏导

## 对 (x) 求偏导：

[
\begin{aligned}
\frac{\partial z}{\partial x} &= f_u \cdot u_x + f_v \cdot v_x \
&= (2u) \cdot y + (\cos(v)) \cdot \left(-\frac{y}{x^2}\right) \
&= 2(xy) \cdot y - \frac{y}{x^2} \cos\left(\frac{y}{x}\right) \
&= 2xy^2 - \frac{y}{x^2} \cos\left(\frac{y}{x}\right)
\end{aligned}
]

## 对 (y) 求偏导：

[
\begin{aligned}
\frac{\partial z}{\partial y} &= f_u \cdot u_y + f_v \cdot v_y \
&= (2u) \cdot x + (\cos(v)) \cdot \frac{1}{x} \
&= 2(xy) \cdot x + \frac{1}{x} \cos\left(\frac{y}{x}\right) \
&= 2x^2 y + \frac{1}{x} \cos\left(\frac{y}{x}\right)
\end{aligned}
]

---

# 5️⃣ 总结计算步骤

1. **识别外函数与内函数**：

   * 外函数：(f(u,v) = u^2 + \sin(v))
   * 内函数：(u(x,y) = xy, v(x,y) = y/x)

2. **求外函数偏导**：(f_u = 2u, f_v = \cos(v))

3. **求内函数偏导**：(u_x, u_y, v_x, v_y)

4. **套链式法则**：
   [
   z_x = f_u u_x + f_v v_x, \quad z_y = f_u u_y + f_v v_y
   ]

5. **代入内函数的表达式**，化简得到最终答案

---

✅ 注意：

* 如果 (f(u,v)) 是抽象函数，就用 (f_u, f_v) 表示偏导
* 如果 (f(u,v)) 是具体函数，就可以 **先求符号偏导，再代入内函数** → 得到具体表达式

# 隐函数的偏导数
这里我其实是计算的时候出了一个问题
如果题目考查我的是高阶隐函数求导的话，我会求完一阶导之后化简一下，
这个似乎是不太对的，感觉会后面二阶导求起来很麻烦

---

# 1️⃣ 问题的本质

假设隐函数：

[
F(x, y) = 0
]

* 一阶导求法：
  [
  \frac{dy}{dx} = -\frac{F_x}{F_y} \quad (\text{这里 } F_y \neq 0)
  ]

* 高阶导求法（例如二阶导）：
  [
  \frac{d^2y}{dx^2} = ?
  ]

你的做法：

1. 一阶导先求出来
2. 化简成一个尽量简单的形式
3. 再去求二阶导

问题：一阶化简可能掩盖了隐函数与 (F) 的结构，导致二阶导求起来复杂或者容易出错。

✅ **正确理念**：

> 高阶隐函数求导时，最好保留原始隐函数 (F(x,y)=0) 的结构，把 (dy/dx) 写成 (-F_x/F_y) 的形式，不要随意化简。

---

# 2️⃣ 二阶导的公式（隐函数法）

一阶：

[
y' = \frac{dy}{dx} = -\frac{F_x}{F_y}
]

二阶导：

[
y'' = \frac{d}{dx} \left( y' \right) = \frac{d}{dx} \left(-\frac{F_x}{F_y} \right)
]

用链式法则展开（记住 (y=y(x))）：

[
y'' = - \frac{d}{dx} F_x \cdot \frac{1}{F_y} + F_x \cdot \frac{d}{dx} \frac{1}{F_y}
]

再代入 (dF_x/dx = F_{xx} + F_{xy} y') 等，最后整理：

[
y'' = -\frac{F_{xx} + 2 F_{xy} y' + F_{yy} (y')^2}{F_y}
]

或者更完整写法：

[
y'' = - \frac{F_{xx} + 2 F_{xy}(-F_x/F_y) + F_{yy}(-F_x/F_y)^2}{F_y}
]

⚠️ 重点：

* 二阶导尽量使用 **一阶导符号形式 (-F_x/F_y)**，不要先算出数值或者过度化简
* 保留隐函数的原始偏导符号 → 推导高阶导方便、整洁

---

# 3️⃣ 几个技巧

1. **不化简一阶导**

   * 化简后可能引入分母、平方、开方 → 二阶导求起来麻烦
   * 保留 (-F_x/F_y) 形式即可

2. **用符号偏导表示**

   * (F_x, F_y, F_{xx}, F_{xy}, F_{yy}) 全用符号
   * 高阶导公式套用符号 → 避免计算混乱

3. **结构清楚，代入方便**

   * 高阶隐函数求导，最终只需要代入一阶导的符号
   * 一阶化简越复杂，二阶导容易错

---

# 4️⃣ 直观理解

* **一阶导** → 切线斜率
* **二阶导** → 曲率
* **保持隐函数形式** → 切线和曲率都从原函数结构直接得到
* **过早化简** → 失去“原函数的结构信息”，后续计算难度增加

---

# 5️⃣ 总结

| 步骤  | 正确做法                                   |
| --- | -------------------------------------- |
| 一阶导 | (-F_x/F_y)，保留符号，不化简                    |
| 二阶导 | 对 (-F_x/F_y) 求导，记住 (y=y(x))，代入一阶导的符号形式 |
| 化简  | 最后才化简，尽量用符号形式保持整洁                      |

## 1️⃣ 方向导数的几何意义

**定义：**
对于多元函数 (f(x, y))，在点 ((x_0, y_0)) 沿着单位方向向量 (\mathbf{u} = (u_1, u_2)) 的方向导数定义为：

[
D_{\mathbf{u}} f(x_0, y_0) = \lim_{h \to 0} \frac{f(x_0 + h u_1, y_0 + h u_2) - f(x_0, y_0)}{h}
]

* **几何理解**：
  方向导数表示在函数图像的某一点，沿着给定方向 (\mathbf{u}) 的瞬时变化率或斜率。

  * 沿着某个方向上升最快 → 最大方向导数
  * 沿着某个方向不变 → 方向导数为 0

---

## 2️⃣ 方向余弦和方向导数的关系

如果方向向量 (\mathbf{u}) 与坐标轴的夹角分别为 (\alpha, \beta)，则：

[
\mathbf{u} = (\cos\alpha, \cos\beta)
]

* 方向导数可以写成：

[
D_{\mathbf{u}} f(x_0, y_0) = f_x(x_0, y_0) \cos\alpha + f_y(x_0, y_0) \cos\beta
]

也就是**偏导数的线性组合**，系数就是方向余弦。

---

方向导数的定义和用方向余弦表示方向导数的式子两个之间为什么成立
这里是用泰勒展开来写的
[
 D_{\mathbf{u}} f(x_0, y_0) = \lim_{h \to 0} \frac{f(x_0 + h u_1, y_0 + h u_2) - f(x_0, y_0)}{h}
 ]

**可以写成**：

[
 D_{\mathbf{u}} f(x_0, y_0) = f_x(x_0, y_0) u_1 + f_y(x_0, y_0) u_2
 ]

也就是偏导数和方向余弦的线性组合。我们一步步来解释原理。

------

## 1️⃣ 方向导数的定义 → 泰勒展开

假设 (f) 在点 ((x_0, y_0)) 可微，可微意味着可以用 **一阶泰勒展开** 近似：

[
 f(x_0 + \Delta x, y_0 + \Delta y) \approx f(x_0, y_0) + f_x(x_0, y_0) \Delta x + f_y(x_0, y_0) \Delta y
 ]

- 这里忽略了高阶无穷小项。

------

## 2️⃣ 沿方向向量 (\mathbf{u} = (u_1, u_2)) 走一点距离 (h)

- 坐标变化是：
   [
   \Delta x = h u_1, \quad \Delta y = h u_2
   ]
- 代入泰勒展开：
   [
   f(x_0 + h u_1, y_0 + h u_2) - f(x_0, y_0) \approx f_x h u_1 + f_y h u_2
   ]

------

## 3️⃣ 除以 (h) 求极限

[
 \frac{f(x_0 + h u_1, y_0 + h u_2) - f(x_0, y_0)}{h} \approx f_x u_1 + f_y u_2
 ]

当 (h \to 0)，高阶无穷小项消失，得到：

[
 \lim_{h \to 0} \frac{f(x_0 + h u_1, y_0 + h u_2) - f(x_0, y_0)}{h} = f_x u_1 + f_y u_2
 ]

这就**严格地把方向导数极限定义**转化成了**偏导数的线性组合**。

------

## 4️⃣ 几何解释

- (u_1, u_2) 是方向余弦 → 描述了方向向量在 (x) 和 (y) 方向的投影
- (f_x, f_y) 是函数在 (x, y) 方向的瞬时变化率
- 于是方向导数 = 在每个坐标轴方向的变化率 × 投影长度之和 → 合成出沿任意方向的变化率

**简洁记忆：**

> “沿方向变化率 = 各坐标方向变化率 × 对应方向的分量之和”


## 3️⃣ 梯度的定义与几何意义

**定义：**

[
\nabla f(x, y) = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right)
]

* **几何意义**：
  梯度是函数在某一点变化最快的方向，其方向就是最大方向导数的方向，大小就是最大方向导数的值。

**与方向导数的关系：**

[
D_{\mathbf{u}} f(x_0, y_0) = \nabla f(x_0, y_0) \cdot \mathbf{u} = |\nabla f| , |\mathbf{u}| \cos\theta = |\nabla f| \cos\theta
]

其中 (\theta) 是梯度方向与方向向量 (\mathbf{u}) 的夹角。

* **结论：**

  * 最大方向导数的大小 = (|\nabla f|)
  * 最大方向导数的方向 = (\nabla f) 的方向
  * 最小方向导数（下降最快）= (-|\nabla f|)，方向与梯度相反
  * 方向导数为 0 → 方向与梯度垂直

---

## 4️⃣ 计算方法总结

1. 求偏导：(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y})
2. 求梯度向量：(\nabla f = (f_x, f_y))
3. 方向导数公式：
   [
   D_{\mathbf{u}} f = f_x u_1 + f_y u_2 = \nabla f \cdot \mathbf{u}
   ]

## 5️⃣ 最大方向导数题目解析

**题目：** “沿着 xx 方向导数取得最大值”

**解释：**

1. 方向导数取得最大值 → 方向与梯度方向一致

   * 公式：(D_{\mathbf{u}} f_{\max} = |\nabla f|)
2. 所以题目问的“xx方向”，要么就是求梯度方向和 xx 方向一致的情况，要么就是要求在 xx 方向上的导数值（用梯度投影）：
   [
   D_{\mathbf{i}} f = \nabla f \cdot \mathbf{i} = f_x
   ]

   * 这里 (\mathbf{i}) 是 x 轴单位向量
   * 如果题目说“沿某方向导数最大”，答案是**沿梯度方向**
   * 如果题目指定“沿 x 方向”，最大值可能小于 (|\nabla f|)，等于梯度在 x 方向上的分量

**几何理解**：

* 梯度就是“坡度最陡的方向”，沿着它走上山最快
* 与梯度垂直走 → 保持高度不变（等高线方向）
